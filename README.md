# 学习计时器 (Study Timer) - MVP版本
本项目为学习时长记录工具，基于HTML/CSS/JavaScript开发，依托GitHub Codespaces完成开发与协作，实现核心计时与本地学习记录存储功能。

## MVP核心功能
1. 基础计时：支持开始、暂停、重置精准计时
2. 科目选择：可选择对应学习科目，匹配学习场景
3. 本地存储：学习记录自动保存至浏览器，刷新/关闭不丢失
4. 记录展示：实时查看历史学习记录

## 运行方式（GitHub Codespaces）
1. 打开团队GitHub仓库：https://github.com/zhan-gitch/study-timer
2. 在Codespaces中加载本项目
3. 打开`index.html`，通过`Open with Live Server`启动
4. 选择科目→点击开始计时→完成后保存记录即可使用

## 技术栈
- 前端基础：HTML5 / CSS3 / JavaScript
- 开发环境：GitHub Codespaces
- 数据存储：浏览器localStorage
